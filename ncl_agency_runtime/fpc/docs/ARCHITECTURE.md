# ARCHITECTURE — FUTURE PREDICTOR COUNCIL

## Purpose
AI-powered prediction and intelligence council. Aggregates signals from markets, tech, and cultural trends to produce forward-looking analysis.

## System Overview
```
[Data Ingestion] --> [Signal Processor] --> [Prediction Engine]
[LLM Agents]    --> [Council Deliberation] --> [Reports/Alerts]
```

## Components
- **Data Ingestion**: RSS, APIs, web scrapers for news/market/tech signals
- **Signal Processor**: NLP classification, trend extraction
- **Prediction Engine**: Multi-model ensemble (LLM + quantitative)
- **Council Agents**: Specialized AI personas debating predictions
- **Report Generator**: Structured forecasts + confidence scores

## Data Flow
External Signals → Ingestion → Processing → Agent Analysis → Council Vote → Output

## Integration Points
- NCL second brain (doctrine storage)
- Matrix Monitor (live display)
- YOUTUBEDROP (content pipeline)

## Key Decisions
- Council model: adversarial agents improve forecast quality
- All predictions versioned and tracked for accuracy scoring
