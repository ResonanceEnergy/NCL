# Active Research

Research requests currently being processed by UNI research cortex.

Each file is a copy of the original queue request with additional fields:
- `assigned_to`: UNI agent handling the research
- `started_at`: ISO timestamp when processing began
- `status`: "in_progress" | "awaiting_review"
- `progress_notes`: Array of timestamped progress entries

Files remain here until research is complete and archived.
