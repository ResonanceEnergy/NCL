# FRACTAL FUTURE — Research Tracks

## Overview
Five research tracks, each examining fractal geometry and entropy through a different lens. Every track feeds the same doctrine, the same evidence pipeline, and the same gate system.

---

## Track 1: Nature Atlas
**Question:** Where do fractals appear in nature, and what entropy problem does each solve?

**Scope:**
- Catalog natural fractal systems (botanical, geological, biological, atmospheric)
- For each: identify the entropy pressure it responds to
- Map the branching pattern to its dissipation function
- Build a visual atlas with annotations

**Key Outputs:**
- Fractal species catalog (min 50 entries)
- Entropy function map per species
- Visual atlas (photos, diagrams, scale comparisons)
- Cross-species pattern analysis

**Doctrine Alignment:** P0 (Entropy Pressure), P1 (Scale Coherence), P6 (Regeneration)
