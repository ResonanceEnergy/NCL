# FRACTAL FUTURE — Prioritization Matrix v1.0

## Scoring (0–5)
- **Impact (I):** Magnitude of benefit if successful
- **Feasibility (F):** Ease and speed of prototyping
- **Evidence (E):** Measurable outcomes available
- **Differentiation (D):** Unique signature; strategic moat

## Formula
```
Priority Score = (I * 0.35) + (F * 0.30) + (E * 0.20) + (D * 0.15)
```

## Tiering Guide
- **Tier A (Start Now):** Score ≥ 4.0
- **Tier B (Build Next):** Score 3.2–3.9
- **Tier C (Long Horizon):** Score ≤ 3.1
