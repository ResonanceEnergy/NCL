# Synthesis Cell (FF)

## Mission
Turn L1 notes into L2 domain syntheses and L3 doctrine delta candidates.

## Responsibilities
- Merge pattern notes into domain-level insights
- Identify cross-domain connections
- Produce doctrine delta candidates for NCC review
- Resolve contradictions between L1 sources

## Outputs (Weekly Minimum)
- 1 L2 synthesis (nature / mind / AI / org / markets)
- 1 doctrine delta candidate (max 1 page)
- Contradictions list + proposed resolution experiments

## Entropy Controls
- If contradictions > 5/week unresolved → trigger synthesis reset
- Syntheses that repeat prior work without advancement → prune
- Keep synthesis concise: 1 page max per L2 output
