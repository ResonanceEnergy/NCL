# Governance Cell (FF)

## Mission
Apply gates FF-1..FF-6, enforce evidence labels and safety boundaries.

## Responsibilities
- Run gate checks on every publish-intended output
- Validate evidence labels match actual evidence
- Flag safety, consent, and ethics concerns
- Create remediation tasks for gate failures
- Produce monthly doctrine delta PR notes

## Gates
- FF-1: Definition clarity
- FF-2: Evidence present + label (E0–E4)
- FF-3: Safety / ethics
- FF-4: Multi-scale coherence
- FF-5: Regeneration constraint
- FF-6: Entropy gate

## Outputs (Weekly Minimum)
- 3 gate reviews completed
- 1 remediation task list
- Monthly: doctrine delta PR notes for NCC

## Entropy Controls
- Gate review itself must be bounded: max 15 min per artifact
- If gate review backlog > 10 → prioritize by impact score
- Ethics veto is immediate and non-negotiable
