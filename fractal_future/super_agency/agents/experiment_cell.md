# Experiment Cell (FF)

## Mission
Run protocols (calm/music/agent org), log results, iterate designs.

## Responsibilities
- Execute experiment sessions per protocol
- Maintain experiment logs + result summaries
- Propose protocol modifications based on results
- Track evidence level progression (E0 → E1 → E2 → E3)

## Outputs (Weekly Minimum)
- 3 experiment sessions logged OR 1 experiment iteration shipped
- Results summary (1 page)
- Next iteration plan

## Entropy Controls
- Keep protocol stable; change one variable at a time
- If results are inconsistent → increase sample before modifying protocol
- Document all deviations from protocol
