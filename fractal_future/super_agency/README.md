# BRS (Bit Rage Systems) — FRACTAL FUTURE Program

## Mission
Build and maintain the FRACTAL FUTURE research engine across NCC, NCL, AAC, and BRS using a cellular agent swarm with local autonomy + global coherence.

## Agent Cells
| Cell | Mission | Weekly Minimum |
|------|---------|----------------|
| Research | Gather patterns, sources, observations → L0/L1 notes | 10 L0, 5 L1, 1 claim |
| Synthesis | Merge L1→L2 domain syntheses + L3 doctrine deltas | 1 L2, 1 delta candidate |
| Experiment | Run protocols, log results, iterate | 3 sessions or 1 iteration |
| Publishing | Convert findings → clean briefs + share packs | 1 brief |
| Governance | Apply gates FF-1..FF-6, enforce evidence labels | 3 gate reviews |
| Entropy Sentinel | Detect drift, coupling, rework; trigger valves | 1 telemetry report |

## Output Cadence
- **Daily:** Signal Brief draft, 1 capture, 1 artifact update
- **Weekly:** Regime Map + cell posture + evidence upgrades + telemetry
- **Monthly:** Doctrine delta PR into NCC + summary synthesis

## Artifact Contract (Every Cell)
Every output must include:
1. Artifact file created/updated
2. 10-line summary
3. Evidence level (E0–E4)
4. Entropy map (ingress/accumulation/valves)
5. Next actions (3)

## Entropy Controls
If entropy indicators rise above budget:
1. Throttle Mode (cut parallelism 50%)
2. Synthesis Reset (freeze new research until L2 ships)
3. Prune & Merge
4. Boundary Reinforcement
5. Gate Re-run (FF-1..FF-6)
