Use `create_issues.sh` to push these to GitHub, or create them manually from `backlog/issues.yaml`.

## Prerequisites

- GitHub CLI (`gh`) installed and authenticated
- Repository: `ResonanceEnergy/NCL`
- Branch: `feat/golden-tasks` or `main`

## Usage

```bash
cd future_predictor_council/scripts
chmod +x create_issues.sh
./create_issues.sh
```
