NEW runtime/intelligence/brief_archiver.py (~460 LOC, 1 module)

  Public API: archive_brief(envelope, pack, mode="am"|"pm", brain)
  Returns: {brief_id, mode, date, md_path, memory_unit_id,
            ideas_registered, elapsed_s}

  Internal pipeline per fire:

  1. _scrub_citations(envelope) — strips placeholder strings
     ("sig_id" / "sig_001" / "sig_042" / "...") from
     trade_ideas[*].sources + top_signals[*].sig_id +
     predictions_watch[*].citations + polymarket_watch[*].citations.
     Real 8-char hex ids pass; everything else is dropped.

  2. _render_md(envelope, mode, brief_id) — builds single
     markdown document:
       ---  (YAML front matter)
       brief_id: 2026-05-31:am:034622
       mode: am
       date: 2026-05-31
       generated_at: ...
       members_succeeded: [macro, pulse, flow, technical]
       council_confidence: 0.78
       n_trade_ideas: 5
       paper_balance_usd: 25000.0
       ...
       ---
       # AM Brief — 2026-05-31
       ## PORTFOLIO  (narrative + structured sub-sections)
       ## INTEL
       ## CALENDAR
       ## JOURNAL
       ## MEMORY
     Persisted to data/morning-brief-md/<date>-<mode>.md.

  3. _persist_to_memory(brain, envelope, md_text, mode, brief_id):
     Creates ONE MemUnit with:
       source         = brief:archive:{am|pm}
       content_type   = brief_snapshot
       content        = full .md text
       importance     = 95 (HIGH — surfaces in working_context)
       tier           = brain
       authority_tier = BRAIN
       metadata = {brief_id, brief_mode, date, generated_at,
                   lanes_present, n_trade_ideas,
                   snapshot_format=markdown_v1,
                   tag=brief:YYYY-MM-DD:{am|pm}}
     Future /memory/search/fused?q="May 31 brief" surfaces it.

  4. _register_trade_ideas(envelope, mode, brief_id) — CRITICAL.
     Walks envelope["lanes"]["portfolio"]["trade_ideas"] and calls
     record_trade_idea_emission for each with:
       source=brief:{am|pm}, ticker, direction, entry/stop/target
       (numeric, with $-stripping fallback), R_per_share,
       planned_qty, stop_type/basis, target_basis, thesis,
       trade_idea_id (auto-generated if missing), and metadata
       including brief_id + brief_mode for retrospective grouping.
     Strategy normalized via risk_governor._normalize_strategy.
     This is the audit's #1 showstopper fix.

    Replaced literal "sig_id" example values with explicit

    NEW Rule 8 making the citation contract explicit:
      "Every sig_id / sources / citations value MUST be replaced
      with the REAL 8-char alphanumeric id= token. NEVER emit the
      literal 'sig_id' / 'sig_001' — those are placeholders. If
      no real id is available, omit the array entirely."

    Most-recent-first listing of archived briefs ({date, mode,

    Returns the .md text directly. iOS can render as markdown
