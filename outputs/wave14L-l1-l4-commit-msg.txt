L4 — PROFIT LADDER (runtime/portfolio/auto_trader/profit_ladder.py)

THE NATRIX-specific "aggressive growing with short dated options then
converting those wins in to long dated options" pattern, encoded.

Flow:
  outcome_attributor.attribute_close fires on stop/target/manual
    ↓
  After bandit + SHAP + drift + friction + prediction-record writes,
  maybe_ladder_from_close checks 4 gates:
    1. ladder enabled (NCL_AT_LADDER_ENABLED=1)
    2. source idempotency (not already laddered)
    3. engine_r ≥ NCL_AT_LADDER_R_THRESHOLD (default +2.0R)
    4. source recipe ∈ short_dated_lottery set
    5. realized_profit ≥ NCL_AT_LADDER_MIN_PROFIT_USD ($50)
    6. destination recipe (leaps_long_dated) enabled
    ↓
  If all pass, emit NEW TradeIdea via tracker:
    source = "profit_ladder"
    strategy = "leaps_long_dated" (configurable)
    ticker = same underlying
    direction = same as closed
    R_per_share = 5% underlying (conservative LEAP premium estimate)
    planned_qty = max(1, ladder_R / R_per_share)
    R-dollars budget = 50% of realized profit (configurable)
    stop_basis = "thesis break OR underlying breaks pre-ladder swing"
    target_basis = "30% underlying move within 120-365 DTE"
    metadata.source_trade_idea_id = original
    metadata.profit_ladder = True
    ↓
  New idea flows through normal loop: governor → calendar gate →
    working_context gate → policy → friction → council quorum (if R≥$1k!)
    → paper open.

Idempotent on source trade_idea_id (eliminated set persisted to
ladder_state.json). Audit JSONL at ladder_emissions.jsonl. MemUnit at
importance 85 documenting the ladder event ("PROFIT LADDER fired:
NVDA long lottery_calls_short_dated closed +3.20R ($640). Rolled 50%
into new leaps_long_dated trade idea abc12345 ($320 R, 120-365 DTE
target)").

Tunables (env, no restart):
  NCL_AT_LADDER_ENABLED=1
  NCL_AT_LADDER_R_THRESHOLD=2.0
  NCL_AT_LADDER_PROFIT_RATIO=0.50
  NCL_AT_LADDER_MIN_PROFIT_USD=50
  NCL_AT_LADDER_DTE_MIN=120
  NCL_AT_LADDER_DTE_MAX=365
  NCL_AT_LADDER_DESTINATION=leaps_long_dated
