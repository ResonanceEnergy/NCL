# composeExecutiveMarkdown helper

Builds .md from the /intelligence/digest payload in the same shape
as the AM/PM .md archives:

  # Executive Brief — YYYY-MM-DD

  ## ⚠️ STALE  (only if brief_age_hours > 12)
  Legacy digest last refreshed Nh ago. Today's Pro Brief substituted.

  ## TODAY'S PRO BRIEF  (or HEADLINE if no fresh substitute)
  **<headline>**
  <portfolio narrative>

  ## NIGHT WATCH
  - status=GREEN
  - key_findings=N
  - recommendations=N

  ## KEY SIGNALS
  - route=HIGH · source=polymarket · title=... · score=0.72
  - route=HIGH · source=reddit · title=... · score=0.68
  - ... (up to 8)

  ## RISK ALERTS
  - <text>
  - <text>

  ## WORKING CONTEXT
  - pinned=true · label=...
  - content=...
  (up to 10)

  ## SOURCE BREAKDOWN
  - polymarket=4
  - reddit=3
  - ...

ProfessionalMarkdownView's block parser handles every line:
  - ## headers get the cyan vertical-bar + rule treatment
  - bullets get cyan dots
  - key=value bullets get split into stacked rows
  - long values (title=, content=) hit the Wave 14DC long-value
    treatment (KEY caption above wrapping body)
  - **bold** renders via AttributedString(markdown:)
