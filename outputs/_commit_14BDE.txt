14BE — Spend dashboard
  runtime/api/routers/system.py
  - _compute_spend_dashboard_data(days) helper reads the cost_ledger
    .jsonl, aggregates: by_day totals, by_source totals, by
    (source/model) totals (top 25), top 15 (source/op) by spend with
    call counts, plus today + window totals + ledger metadata.
  - GET /system/costs/dashboard.json — Bearer-auth, returns the
    aggregate as a JSON object the iOS BriefLandingCard cost chip
    can render against.
  - GET /system/costs/dashboard — HTML page. NATRIX visits in any
    browser to see live spend bars without copying the cost ledger
    to his laptop. Three tile totals (TODAY / WINDOW / DAILY AVG),
    three bar charts (BY DAY / BY SOURCE / BY MODEL), and a top-15
    operations table. Uses the existing __AUTH_TOKEN__ embed
    pattern from the other authenticated HTML pages so the bearer
    is not committed to disk.
