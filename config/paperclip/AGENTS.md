# NARTIX Agents

## Claude-Chair
Permanent council moderator. Uses advanced multi-LLM collaboration strategies (debate tournament, meta-reasoning, simulated annealing, etc.). Spawns councils, runs hybrid Copilot coding loop with Claude Opus 4, and enforces MWP folder structure. Has Computer Use for VS Code automation.

## Researcher
Perplexity-powered investigation agent. Sources claims, fact-checks council assertions, provides cited research briefs. Primary agent for research_investigation workflows.

## TrendsAnalyst
Gemini-powered for YouTube, Google Trends, and News intelligence. Identifies emerging patterns, market signals, and content trends. Primary agent for trend_analysis workflows.

## FirstStrike
Grok-powered mobile intelligence agent. Receives and pre-processes pump prompts from iPhone. Formats raw NATRIX intent into structured council-ready briefs with X context.

## Engineer
Claude-powered execution support. Runs in 03-Execution alongside Claude-Chair. Handles test execution, code review, debugging analysis, and deployment verification.

## LocalThinker
Ollama qwen3:8b for fast local reasoning. Zero-latency tasks that don't need cloud APIs. Code review, quick analysis, format checking. Runs entirely on Mac Mini M4 Pro.
