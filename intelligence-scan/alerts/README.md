# Intelligence Alerts

Real-time anomaly alerts generated by Awarebot-FPC scanner.

## File Format

Each alert is a JSON file named `alert-{YYYYMMDD-HHMMSS}-{source}.json`:

```json
{
  "alert_id": "alert-20260404-143022-x",
  "source": "x | youtube | reddit | polymarket | rss",
  "severity": "CRITICAL | HIGH | MEDIUM | LOW",
  "category": "geopolitical | market | tech | alt-science | sentiment",
  "title": "Brief alert title",
  "summary": "What was detected and why it matters",
  "raw_signal": {
    "url": "https://...",
    "author": "@handle",
    "content_preview": "First 500 chars...",
    "engagement_metrics": {}
  },
  "related_scenarios": [],
  "recommended_action": "escalate | monitor | archive",
  "created_at": "2026-04-04T14:30:22Z",
  "acknowledged": false,
  "escalated_to": null
}
```

## Escalation Rules

- CRITICAL: Immediately forwarded to NCL mandate generation
- HIGH: Queued for next council session
- MEDIUM: Added to daily intelligence briefing
- LOW: Logged to signals/ for pattern detection
